Job Application Software

Technologies Used

  • Python Programming
  • Python Tkinter
  • Sql Database
  • Adzuna Api

Description

The main focus of this application was on its functionality especially on authentication of users and saving their personal data to the database.

This desktop application was inspired by the struggles faced by job seekers of all kinds from developers to drivers on a day to day basis. I built it to make it easier for employees to reach employers. It is very basic, with a sign up page, login page, data entry page, jobs page and a selected jobs page.

How the application works

Sign up

To sign in, a user has to fill the form below to completion. The data is then saved in a sql database and the password encripted.

Log in

Once a user has an account activated, loging in is simple.
Input the names and password used to create the account. If the password is correct, the user is directed to the main page, but if not, an arror is displayed.

Main page

The main page gets it's data from the internet through the ADZUNA API. Jobs are displayed for the user to select one of their choice. When a user selects a job, it is recorded like in a shopping cart.

View selected job

To view the jobs selected from the main page, the user has to navigate to this page. Here, the user can pick a job to apply or remove selected jobs. On application, they are redirected to the data entry page.

Enter personal information

For the job seeker apply and to be identified by the employee, they have to fill in this form. The data entered by the user is saved to a database and will be shared with the employee.
